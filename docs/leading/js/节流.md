# 节流

- 在一段特定的时间内，只执行一次

```js
function throttle(fn, delay) {
  let timer;
  return (...args) => {
    if (!timer) {
      timer = setTimeout(() => {
        fn.apply(this, args);
        timer = null;
      }, delay);
    }
  };
}
const s = () => {
  console.log("123");
};

document.querySelector("button").addEventListener("click", throttle(s, 1000));
```
