# 如何解决跨域问题

- **vite 代理转发**

```js
 server: {
    proxy: {
        '/api': {
            target: '代理地址',
            changeOrigin: true,
            rewrite: (path) => path.replace(/^\/api/, '')
        }
    }
}
```

- **jonsp**

- **CORS（跨域资源共享**

  - 这是最直接的解决方案。在服务器端设置响应头 Access-Control-Allow-Origin，允许前端页面从不同的源发送 XMLHttpRequest 请求。

- **nginx 反向代理**

```conf

// 通过 nginx.conf 文件

localtion /api/ {
        proxy_pass  http://xxx:3000
}


```
